Virtual Cell

A sandbox for hypothesis-driven discovery: simulate gene expression perturbations and observe their effects on cellular systems.

Launch Virtual Cell →

Try the interactive demo

Cell Type Selection

Choose from available cell types to simulate perturbations in different cellular contexts. Each cell type has its own baseline expression profile.

Gene Knockout Simulation

Search and select any gene to perturb. The model predicts how knocking out that gene affects the expression of all other genes in the cell.

Cell State Projection

Visualize the predicted shift in cell state with a UMAP projection. See exactly how the perturbation moves the cell relative to the control population.

Expression Analysis

Explore results in a sortable, searchable table showing predicted changes across all genes. Filter to see only affected genes and identify the most impacted pathways.
